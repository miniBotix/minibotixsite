What is a Push Button?

A push button is a simple switch used to control electrical circuits. When pressed, it completes a connection; when released, it breaks the connection. It’s commonly used in embedded systems to trigger actions like starting/stopping a process.

Connection: Connect one side to GND and the other to a digital input pin with a pull-up resistor (or use INPUT_PULLUP mode).

Code: Push Button to Turn ON a Buzzer

#define button 2
#define buzzer 9

void setup() {
  pinMode(button, INPUT_PULLUP);
  pinMode(buzzer, OUTPUT);
}

void loop() {
  if (digitalRead(button) == LOW) {
    digitalWrite(buzzer, HIGH);
  } else {
    digitalWrite(buzzer, LOW);
  }
}
➤ When button is pressed → Buzzer turns ON
➤ When button is released → Buzzer turns OFF

Understanding Loops in C++

Loops let you run a block of code multiple times. Common loops include:

  • for loop – used when number of iterations is known
  • while loop – runs as long as a condition is true

DHT11 Reading Using for Loop

Note: You need the DHT.h library installed.

#include <DHT.h>

#define DHTPIN 2
#define DHTTYPE DHT11
DHT dht(DHTPIN, DHTTYPE);

void setup() {
  Serial.begin(9600);
  dht.begin();
}

void loop() {
  for (int i = 0; i < 5; i++) {
    float temp = dht.readTemperature();
    float hum = dht.readHumidity();

    Serial.print("Reading ");
    Serial.print(i + 1);
    Serial.print(": Temp = ");
    Serial.print(temp);
    Serial.print(" °C, Humidity = ");
    Serial.print(hum);
    Serial.println(" %");
    
    delay(2000); // wait 2 seconds
  }

  while (1); // Stop loop after 5 readings
}

int i = 0;

void setup() {

}

void loop() {
  while (i < 3) {
    Serial.print("Loop #");
    Serial.println(i + 1);
    i++;
    delay(1000);

Introduction to Sensors

Sensors are devices that detect changes in the environment and convert them into electrical signals.

15 Basic Sensors, Input Voltage & Applications:

  • DHT11 – Temp & Humidity | 3.3V/5V | Weather monitoring
  • LM35 – Temperature | 5V | Room temp, fire detection
  • PIR Sensor – Motion | 5V | Security, automation
  • IR Sensor – Obstacle | 3.3V/5V | Line following robot
  • Ultrasonic Sensor (HC-SR04) – Distance | 5V | Range finding
  • LDR – Light | 5V | Auto lighting
  • MQ-2 – Gas (LPG, Smoke) | 5V | Air quality
  • Soil Moisture – Moisture | 3.3V/5V | Smart irrigation
  • Rain Sensor – Rain detection | 5V | Weather station
  • Flame Sensor – Fire | 3.3V/5V | Safety systems
  • Vibration Sensor – Motion | 3.3V/5V | Vibration monitoring
  • Hall Effect Sensor – Magnetic field | 3.3V/5V | RPM, contactless switches
  • Sound Sensor – Noise | 5V | Clap switch
  • Pulse Sensor – Heartbeat | 3.3V/5V | Health monitoring
  • UV Sensor – UV radiation | 3.3V | Sunlight detection
