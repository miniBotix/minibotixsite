🔥 Project 001: Smart Fire Alarm System

This project detects fire using a flame sensor and triggers a buzzer alarm. It is commonly used in early fire alert systems for homes or labs.

🧰 Components Used:

  • Flame Sensor
  • Buzzer
  • Arduino Uno
  • Resistors, Jumper wires

📦 Libraries Used:

  • No external libraries required – basic Arduino functions

🧑‍💻 Code:

// Pin Definitions int flameSensor = 2; // Flame sensor connected to digital pin 2 int buzzer = 8; // Buzzer connected to digital pin 8 void setup() { pinMode(flameSensor, INPUT); // Set sensor pin as input pinMode(buzzer, OUTPUT); // Set buzzer as output Serial.begin(9600); // Begin serial communication } void loop() { int fireState = digitalRead(flameSensor); // Read sensor value if (fireState == LOW) { // LOW means fire detected Serial.println("🔥 Fire Detected!"); digitalWrite(buzzer, HIGH); // Turn on buzzer } else { Serial.println("✅ No Fire."); digitalWrite(buzzer, LOW); // Turn off buzzer } delay(1000); // Wait 1 second }

💧 Project 002: Soil Moisture Sensor Based Watering System

This project waters a plant automatically by checking soil dryness using a moisture sensor. If moisture is low, water is poured using a pump.

🧰 Components Used:

  • Soil Moisture Sensor
  • Relay or Water Pump
  • Arduino Uno
  • Jumper wires

📦 Libraries Used:

  • No external libraries – uses analogRead and digitalWrite

🧑‍💻 Code:

int moisturePin = A0; // Sensor connected to A0 int pumpPin = 7; // Water pump or relay connected to pin 7 int threshold = 400; // Adjust threshold value based on testing void setup() { pinMode(pumpPin, OUTPUT); Serial.begin(9600); } void loop() { int moisture = analogRead(moisturePin); Serial.print("Moisture Level: "); Serial.println(moisture); if (moisture < threshold) { Serial.println("💧 Soil Dry - Pump ON"); digitalWrite(pumpPin, HIGH); } else { Serial.println("🌱 Soil Moist - Pump OFF"); digitalWrite(pumpPin, LOW); } delay(2000); }

🖥️ Serial Monitor Output:

💧 Soil Dry - Pump ON

Moisture Level: 768
🌱 Soil Moist - Pump OFF
